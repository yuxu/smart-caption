A WordPress plugin that provides a custom block for figure and figcaption elements in Gutenberg editor.

== Description ==

This plugin provides a universal figure block that allows users to add captions to various content types like lists, quotes, tables, etc., while maintaining proper HTML semantics.

Features:
* Custom Gutenberg block for figure elements
* Configurable figcaption position (above or below content)
* Admin settings to whitelist allowed inner blocks
* Basic styling for captions

== Installation ==

1. Upload the `smart-caption` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The block will be available in the Gutenberg editor.

== Frequently Asked Questions ==

= How do I use the block? =

After activating the plugin, you'll find the "Universal Figure" block in the Gutenberg editor. Add your content inside and set a caption.
